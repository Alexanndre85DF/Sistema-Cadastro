# Sistema de Cadastro

Este é um sistema de cadastro de clientes desenvolvido em Python.

## Instalação

1. Instale o Python 3.8 ou superior
2. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```
3. Execute o script de build:
   ```
   python build.py
   ```

## Uso

Após a instalação, o executável será criado na pasta `SistemaCadastro`. Basta executar o arquivo `SistemaCadastro.exe` para iniciar o programa.
